# 俄语剪贴板朗读器 - 重音标注+形态学分析版
利用python编写的一款智能的俄语学习辅助工具，能够自动监控剪贴板中的俄语文本，并实现语音朗读、重音标注、中文翻译和形态学分析等功能。

## 🌟 主要功能
### 1. 智能剪贴板监控
自动检测剪贴板中的俄语文本

可自定义检查间隔和冷却时间

智能过滤非俄语内容

### 2. 高质量语音朗读
使用Google TTS引擎生成俄语语音

支持自定义朗读次数和间隔时间

音频增强功能，提高播放音量

兼容macOS、Windows和Linux系统

### 3. 专业俄语分析
重音标注：自动为俄语单词标注重音位置

中文翻译：俄语到中文的实时翻译

形态学分析：分析单词的词性、格、数等语法特征

单词历史记录：自动保存分析过的单词原形和翻译

### 4. 智能单词管理
自动提取处理过的俄语单词

按单词原形存储，避免重复

支持导出为TXT格式单词卡

三种导出模式：最近一次、历史所有、日志提取

### 5. 用户友好界面
现代化GUI界面，支持深色模式

实时显示处理结果和日志

参数配置面板，所有设置可自定义

标签页布局，功能分区清晰
